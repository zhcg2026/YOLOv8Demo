# 城市问题图像采集指南

本文档用于指导采集员拍摄城市管理问题的训练照片，请仔细阅读并按要求执行。

---

## 一、基本要求

### 图片质量
| 要求 | 说明 |
|------|------|
| 分辨率 | 至少 640×640，建议 1280×720 以上 |
| 清晰度 | 避免模糊，手稳或使用防抖功能 |
| 光线 | 避免过暗或过曝，顺光拍摄效果更好 |
| 格式 | JPG 格式，不需要压缩处理 |

### 拍摄原则
- **多样性强**：不同角度、距离、场景、时间
- **特征清晰**：问题类型要一眼能辨认
- **避免重复**：同一地点连续拍多张只算1张有效数据

---

## 二、单体垃圾采集

### 需采集的垃圾类型
- 塑料袋、饮料瓶、外卖盒、纸屑、烟头
- 建筑垃圾、落叶堆积、生活垃圾堆等

### 拍摄要求
- **距离**：近距离（1-3米）和中距离（5-10米）都要采集
- **角度**：正面、俯视为主
- **背景**：街道、人行道、小区地面、公园草地等
- **状态**：单个垃圾、散落垃圾、堆积垃圾

### 示例场景
```
✅ 街边散落的塑料袋
✅ 人行道上的饮料瓶
✅ 非机动车道上的纸屑
......

## 三、占道经营采集

### 需采集的占道类型
- 流动摊贩（水果摊、小吃摊、早餐摊）
- 店外经营（商铺货物摆到人行道）
- 违章广告牌、遮阳棚占道


### 拍摄要求
- **距离**：中远距离（5-15米），能看清整体场景
- **角度**：正面为主，能体现"占道"特征
- **关键**：要拍到被占用的道路/人行道区域
- **程度**：轻微占道、严重占道都要采集

### 示例场景
```
✅ 人行道上的早餐摊位
✅ 商店门口摆放的货物占用人行道
✅ 街边的水果流动摊贩
......

---

## 四、背景/负样本采集

除了问题场景，也需要采集**正常场景**（没有问题的街道、小区等），用于训练模型识别"无问题"的情况。

### 采集内容
- 干净整洁的街道、人行道
- 正常营业的商铺（无占道）
- 清洁的小区、公园环境

### 数量要求
约占总采集量的 **20-30%**

---

## 五、采集数量建议

| 类型 | 建议数量 |
|------|----------|
| 单体垃圾 | 500-1000 张 |
| 占道经营 | 500-1000 张 |
| 背景/负样本 | 200-300 张 |

## 七、注意事项

### ⚠️ 隐私保护
- 避免拍到清晰的人脸
- 避免拍到车牌号码
- 如拍到敏感信息，请删除或遮挡

### ⚠️ 安全第一
- 拍摄时注意自身安全
- 不要影响交通秩序
- 不要进入危险区域

### ⚠️ 禁止行为
- 不要重复拍摄同一地点
- 不要编造或摆拍虚假问题
- 不要拍摄与任务无关的内容

---

## 八、常见问题

**Q: 一个垃圾可以拍多张吗？**
A: 可以，但要有角度、距离的变化，重复角度的只计1张。

**Q: 拍到人脸怎么办？**
A: 尽量避开，如果不可避免，后期标注时会处理。

**Q: 雨天可以采集吗？**
A: 可以，不同天气条件能增加数据多样性。

如有疑问，请及时反馈。谢谢配合！
